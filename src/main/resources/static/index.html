API Reference
All REST endpoints · Spring Boot · port 8080
Products
GET/api/productsList all (paged)
GET/api/products/{id}Get by ID
GET/api/products/by-category?category=Filter by category
GET/api/products/groupedGrouped by category · Stream + TreeMap
GET/api/products/inventory-valueTotal value · mapToDouble + reduce
GET/api/products/low-stock-report?threshold=Low-stock report · Direct JDBC
POST/api/productsCreate product
PUT/api/products/{id}Update product
PUT/api/products/{id}/stock?quantity=Update stock qty
DEL/api/products/{id}Delete product
Orders
GET/api/ordersList all orders
GET/api/orders/{id}Get by ID
GET/api/orders/groupedGrouped by status · Stream
GET/api/orders/revenueDelivered revenue · filter + sum
GET/api/orders/revenue-reportRevenue by status · Direct JDBC
POST/api/ordersPlace order · Builder + Strategy + Lock
PUT/api/orders/{id}/status?status=Update status
Suppliers
GET/api/suppliersList all
GET/api/suppliers/{id}Get by ID
POST/api/suppliersCreate supplier
PUT/api/suppliers/{id}Update supplier
DEL/api/suppliers/{id}Delete supplier
Warehouses
GET/api/warehousesList all
GET/api/warehouses/{id}Get by ID
GET/api/warehouses/stock-check?sku=&qty=Parallel check · ExecutorService
POST/api/warehousesCreate warehouse
DEL/api/warehouses/{id}Delete warehouse
